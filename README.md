 Bütçe Takip Uygulaması (Budget Tracker)

JavaScript, DOM Manipülasyonu ve LocalStorage kullanılarak geliştirilmiş, modern ve kalıcı hafızaya sahip bir kişisel finans uygulaması.

 Özellikler

* **Gelir/Gider Ekleme:** Harcama türüne göre (+/-) otomatik işaretleme ve renklendirme.
* **Anlık Bakiye Hesabı:** Eklenen veya silinen verilere göre toplam tutarın anında güncellenmesi.
* **LocalStorage (Kalıcı Hafıza):** Sayfa yenilense bile verilerin kaybolmaması.
* **Akıllı Validasyon:** Boş veri girişinin engellenmesi.
* **Dinamik DOM:** Liste elemanlarının HTML yazmadan, tamamen JavaScript ile oluşturulması.

Kullanılan Teknolojiler

* HTML5
* CSS3 (Flexbox & Responsive Tasarım)
* Vanilla JavaScript (ES6+)
